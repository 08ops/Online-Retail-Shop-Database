# Online Retail Store Database

CPEN 211 – Database System Design

## Project Description
This project implements a relational database system for an online retail store using PostgreSQL.

The database manages:
- Customers
- Products
- Orders
- Order Items
- Shipping

## Features
- ER diagram design
- PostgreSQL schema implementation
- Normalized database structure
- Example SQL queries

## Technologies
- PostgreSQL
- SQL
- ER Modeling
